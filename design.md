# Guard AI – Design

## Overview
Guard AI is a simple AI system that checks if a user’s financial decision is risky. Instead of judging, it gives silent warnings and safer suggestions.

## How it works
1. User enters a plan (like investing or borrowing money)
2. AI checks if it looks risky
3. System gives a risk level
4. User sees warning and options

## Main Parts

### Interface
A simple mobile or web app where users type their plans or questions.

### AI Logic
- Understand user intent
- Compare with known risky patterns
- Generate risk score

### Data
- Scam examples
- Financial risk patterns
- Public fraud reports

## Privacy
User data stays private. No public sharing.

## Future Ideas
- Regional language support
- Voice input
- More personalized guidance
